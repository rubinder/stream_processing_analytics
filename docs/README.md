# docs/

Project documentation — design specs, implementation plans, and other notes generated as part of the brainstorming → planning → execution workflow.

## Layout

- `superpowers/` — artifacts produced by the [`superpowers`](https://github.com/anthropics/superpowers) brainstorming/planning skills.
  - `specs/` — design specs (the **what** and **why**).
  - `plans/` — implementation plans (the **how**, broken into bite-sized TDD tasks).

The project's quick-start lives in the repo-root `README.md`, not here.
